    An object representing an active splash, such as a blood particle.

    Properties

    age

    age

    The splash's current age, measured in seconds.

    Returns:

    • result (number)

    maxAge

    maxage

    The splash's maximum age, measured in seconds.

    Returns:

    • result (number)

    node

    node

    Read-only. The splash's node.

    Returns:

    • result (niNode)
